> File Deep Info is a professional-grade file forensics and deep inspection tool for macOS. Go beyond what Finder shows you — uncover hidden data, verify true file formats, extract embedded objects, and audit metadata with surgical precision.

> Whether you're a developer debugging file structures, a data analyst inspecting metadata, a journalist protecting sources, or simply a privacy-conscious user — File Deep Info gives you the full picture.
 
> ▸ TRUE FORMAT IDENTIFICATION
> Stop trusting file extensions. Our triple-engine detection system combines MIME/Magic Number analysis, macOS native UTI framework, and 97 deep signature scanners to reveal a file's real identity. Detect disguised executables (e.g., mismatched extensions), identify extension-less files, and verify format integrity for CAD (DWG/DXF), encrypted archives, iWork, and thousands more.
 
> ▸ HIDDEN DATA DISCOVERY
> Files carry more than what meets the eye. File Deep Info exposes:
> • Office (Word/Excel/PPT): Hidden rows, sheets, white-on-white text, comments, revision history, VBA macros, master slide content
> • PDF: Embedded attachments, hidden OCG layers, transparent/zero-size text, fake redactions, JavaScript
> • Images (JPG/PNG/GIF): EXIF GPS coordinates, camera info, EOF trailing data, LSB steganography
> • Archives (ZIP/TAR/7Z): Nested executables, hidden payloads, overlay/shadow data
> • Apple iWork (Pages/Numbers/Keynote): Internal data resources, embedded media, property lists
 
> ▸ EMBEDDED OBJECT EXTRACTION
> Dig into any document's internal structure. Extract OLE objects from Office files, multimedia assets, PDF attachments, iWork internal resources, and trailing shadow data appended after EOF markers. Pro users get unlimited extraction with recursive analysis — drill down through infinite nesting levels (Word → Excel → ZIP → Image) with anti-spoofing identification that ignores extensions and reads raw signatures.
 
> ▸ METADATA AUDIT
> Full metadata extraction for Office documents (author, edit history, app version, word count), images (complete EXIF/TIFF with GPS, camera model, shooting parameters), and macOS-specific extended attributes including download source URLs (kMDItemWhereFroms) and quarantine flags.
 
> ▸ CHECKSUM & HEX ANALYSIS
> One-pass calculation of MD5, SHA1, SHA256, SHA512 & CRC32 with memory-mapped I/O for large files. Instant 256-byte hex header preview with ASCII correlation for quick signature identification. All processing runs on background threads — zero UI lag guaranteed.
 
> ▸ SECURITY & PRIVACY FOCUSED
> • Detect format spoofing and correct file extension mismatches
> • Identify steganographic data hidden in images
> • Inspect embedded PDF JavaScript for specific logic execution
> • Reveal "fake redactions" that only visually hide sensitive text
> • Track file origins through macOS extended attributes
